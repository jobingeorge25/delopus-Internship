Colors in HTML

Colors in HTML are represented as either values(rgb,hex,hsl) or by their standard names.

HTML supports 140 standard color names like :

white

and

chocolate

        RGB stands for the amount of red, blue, and green components in the color, values range from 0-255
        This text uses rgb values to assign the color
        HEX refers to the hex code that represents the same rgb values but in hexadecimal system( the hex equivalent of 0-255 is 0 to ff) 
        This text uses hex values to assign the color
        HSL refers to hue(pure color), saturation(intensity of color), lightness(amount of light on color)
        This text uses hsl values to assign the color
