Block and Inline

Block elements take entire width of page (Eg- div,p)
Inline elements only take the required amount of space(Eg-span,strong)

Div and Span

'div' is used as a container for other elements
'span' is used as an inline container for text

Class and ID

Class

  • Is used to assign a specific name to an element which it can be addressed by
  • By specifying .classname in styles we can assign properties just to that specific element
  • Various elements can belong to the same class
  • element can belong to multiple classes as well
  • Classes can be used for many functions in javascipt

Id - It is used to uniquely identify an element.
