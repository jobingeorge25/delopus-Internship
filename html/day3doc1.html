I Introduction

Links are defined using the <a> tag

  • href attribute provides the link
  • target specifies how the link will be opened

By default links provided are opened in the same browser window

Example

The target atttribute has the following options

  1. self - default
  2. blank -opens in new window/tab
  3. parent -opens in new window/tab
  4. top -opens in new window/tab

We can insert an img element within a link instead of text

we can use mailto: within href to send mail to the given id
