Images

I Intro

Images are actually just placeholders for the image which is then referenced by a link

The two basic attributes of the image element are:

  • src- to specify source/path if image
  • alt- displays the given text incase there is an image in loading the image

We can specify the dimensions of an image using either

  • width and height attributes
  • using style

Using the style attribute is preferred so that sty;e sheets will not be able to change the image
We can use float attribute to make an image appear next to our text


II Image Map

An image map is a map in which we can define clickable areas

Steps to create a map:

  1. add usemap attribute to the image that will serve as the map
  2. create map element
  3. define areas using area element and specify coordinates
  4. add link in area element

We can use the following shapes :rect,circle,poly,default
This concept has been used on the image with the html,css and js logos
Clicking on any of them takes you to their respective wikipedia site

III Background Images

We can use as an image as the background for an element or even for the entire page
For an element we may either specify it in the style attribute or or specify it in head
To assign a background image for the entire page it must be specified within the body

IV Picture element

The picture element is used to provide multiple sources for the same image
We can use this to select the most suitable image based on the device of the user
This feature can also be used to provide different formats of the same image incase one is not suppported
It is recommended to use the <img> attribute at the end of all the different options
This image will be used if <picture> attribute isnt supported by the browser or if none of the other sources are supported

V Favicon

A small image displayed on the title bar is called the favicon
To add a favicon:

  1. add a link element in the head section
  2. rel attribute must be assigned to 'icon'
  3. type attribute -"image/x-icon"
  4. then specify href
